# MyBirthdayReminderBot

Telegram-бот для хранения дней рождения друзей и напоминаний за 1 день до праздника.

## Команды
- "/start" — Описание функционала бота
- "/add"` — Добавить день рождения
- "/list" — Посмотреть список
- "/delete" — Удалить запись

## Как запустить
1. Установите зависимости:
pip install pyTelegramBotAPI
2. Получите токен у @BotFather: Перейдите в Telegram → найдите бота @BotFather → создайте нового бота → скопируйте токен.
3. Откройте файл bot.py и замените строку: TOKEN = 'YOUR_BOT_TOKEN_HERE' на ваш реальный токен, например: TOKEN = '123456789:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw'
4. Запустите бота: python bot.py
